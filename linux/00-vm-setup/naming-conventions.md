# Convenciones de nomenclatura — Homelab

## VMs y hosts
Formato: hl-lab-deb12-01
- hl: homelab
- lab: laboratorio
- deb12: debian 12
- 01: máquina 1

## Snapshots
Formato: NN-descripcion-corta

## Justificación
La nomenclatura es importante para categorizar cada MV de forma correcta y saber qué hace cada una.
